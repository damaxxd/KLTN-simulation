# Project Overview

## Problem
We study QoE optimization in a CR-RSMA uplink system with UAV users transmitting SVC video.

## Key Idea
Combine:
- RSMA (efficient spectrum sharing)
- SVC (layered video adaptation)
- QoE-aware optimization

## Contributions
1. Cross-layer PHY + video design
2. SVC-aware power allocation
3. QoE-driven optimization (not just rate)

## Expected Results
- QoE vs SNR curves
- Outage probability vs SNR
- Power allocation behavior
